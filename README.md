# Employee Gender & Salary Analytics (SQL + Tableau)

## Project Overview
This project analyzes employee demographics, management distribution, and salary trends using a relational employee database. SQL was used to perform analytical queries and create a stored procedure, while Tableau was used to visualize insights through interactive dashboards.

## Tools & Technologies
- SQL (MySQL)
- Tableau
- Stored Procedures
- Data Modeling & Aggregations

## Key Analysis
1. Gender distribution of employees by year (since 1990)
2. Male vs female managers by department and year
3. Average salary comparison by gender with department-level filters
4. Stored procedure to calculate average salary by gender within a custom salary range
